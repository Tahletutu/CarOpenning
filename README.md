# TurboBox Garage

Mini-jeu HTML/CSS/JS de box opening sur le thème des voitures.

## Fonctionnalités

- Ouvrir 3 types de box avec animation de pack opening.
- Dropper des voitures complètes ou des pièces.
- Monter / remplacer / retirer des pièces sur une voiture.
- Vendre les voitures et les pièces.
- Combiner 3 pièces du même type et de la même rareté pour obtenir une pièce meilleure.
- Mini-jeux pour gagner de l'argent : départ parfait, atelier express, drift timing.
- Sauvegarde automatique dans le `localStorage` du navigateur.

## Lancement

Ouvre simplement `index.html` dans un navigateur moderne.

Aucune dépendance, aucun serveur obligatoire.
